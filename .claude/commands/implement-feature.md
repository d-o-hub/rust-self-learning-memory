Implement a new feature systematically.

Guide through the complete feature implementation process from planning to completion.

Usage: /implement-feature [feature description]

Steps:
1. Understand and clarify requirements
2. Design the feature architecture
3. Plan module structure (keeping files < 500 LOC)
4. Implement types and data structures
5. Implement core logic
6. Add storage layer (Turso + redb)
7. Create public API
8. Write comprehensive tests (unit + integration)
9. Add documentation with examples
10. Run quality checks (fmt, clippy, test)
11. Verify everything works
12. Create commit with clear message
13. Report completion

Use the feature-implementer agent for this workflow.
