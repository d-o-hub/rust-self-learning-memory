### File Naming Convention

**Active Documents**:
- `CATEGORY_DESCRIPTION_DATE.md` (e.g., GAP_ANALYSIS_REPORT_2025-12-29.md)
- `CATEGORY_STATUS.md` (e.g., CONFIGURATION_OPTIMIZATION_STATUS.md)
- Uppercase for major documents

**Subdirectories**:
- Lowercase with hyphens (e.g., benchmark_results/, test-reports/)
- Organized by category

**Archived Documents**:
- Moved to archive/ with original name
- Organized by date or category

### Content Requirements

**All Active Documents Must Have**:
- **Date**: Generated/last updated date
- **Status**: Current state (Active, Complete, Planned)
- **Purpose**: Clear statement of what the document covers
- **Cross-References**: Links to related documents
- **Version**: If version-specific

**Status Documents Must Have**:
- **Executive Summary**: High-level overview
- **Metrics**: Quantifiable progress indicators
